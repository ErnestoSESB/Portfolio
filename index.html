Meus Projetos

SmartFeeder

O SmartFeeder é um sistema de alimentação automática para animais desenvolvido como projeto de TCC. Combinando hardware e software, o alimentador oferece uma solução completa para o cuidado automatizado da saúde alimenticia de animais de diferentes portes.


Utilizando tecnologias de Internet das Coisas (IoT), o sistema desenvolvido em C++ permite que tutores programem horários de alimentação, controlem porções e monitorem o consumo alimentar remotamente através de um aplicativo intuitivo desenvolvido em Python através do Framework Django.

App de Receitas Culinárias

Aplicativo web para gerenciamento e descoberta de receitas culinárias. Desenvolvido com foco em usabilidade e experiência do usuário, permitindo que usuários salvem, compartilhem e explorem receitas de forma intuitiva.

ENEM Cronograma - Organizador de Estudos

Ferramenta de planejamento de estudos voltada para estudantes que se preparam para o ENEM. O sistema permite criar cronogramas personalizados, acompanhar o progresso e organizar conteúdos por área de conhecimento.
